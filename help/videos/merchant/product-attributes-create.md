# 製品属性の作成

Adobe Commerceは、属性を使用して、製品、顧客および製品の評価に関連する情報を管理します。 属性とは、フォーム、レポート、製品レビュー、レイヤーナビゲーション、電子メールメッセージのフィールドとして表示される情報です。 これらは商品カタログの構成要素であり、商品の特定の特性を表します。

## このビデオは誰のためのものですか？

- Web サイトマネージャー
- e コマースマーチャンダイザー
