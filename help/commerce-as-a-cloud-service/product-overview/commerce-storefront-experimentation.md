# 組み込みのテスト機能でストアフロントを最適化する

Adobe Commerce as a Cloud Serviceを利用して、A/B テストを実施し、開発者やサードパーティのツールに頼ることなく、ストアフロントコンテンツを最適化する方法をご確認ください。 組み込みのテスト機能を使用して、コンバージョンとエンゲージメントに対する効果を測定する方法を紹介します。

## この動画は誰のためのものでしょうか？

* e コマースマネージャー
* デジタルマーケティングスペシャリスト
* Conversion Optimization Analyst

## ビデオコンテンツ

* Adobe Commerceで直接A/B テストを設定および実行して、ストアフロントのパフォーマンスを向上させる方法を説明します。
* ビルトインのテストと製品レコメンデーションが、コンバージョンとエンゲージメントの向上にどのように役立つかをご確認ください。
