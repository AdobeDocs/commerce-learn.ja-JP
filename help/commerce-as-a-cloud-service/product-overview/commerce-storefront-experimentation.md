# 組み込み実験によるストアフロントの最適化

Adobe Commerce as a Cloud Serviceを使用することで、ビジネスチームは開発者やサードパーティのツールに依存することなく、A/B テストを実行し、ストアフロントコンテンツを最適化できます。 ネイティブの実験機能を使用して、コンバージョンとエンゲージメントに対する影響を測定する方法を説明します。

## このビデオの目的は誰ですか。

* E コマースマネージャー
* デジタルマーケティング担当者
* コンバージョン最適化アナリスト

## ビデオコンテンツ

* Adobe Commerceで直接 A/B テストを設定および実行して、ストアフロントのパフォーマンスを向上させる方法について説明します。
* 組み込みの実験と製品のレコメンデーションが、コンバージョンとエンゲージメントを向上させる方法を説明します。
