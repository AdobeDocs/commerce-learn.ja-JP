# コンポーザビリティを備えた柔軟なコマースエクスペリエンスの構築

Adobe Commerce as a Cloud Serviceが、俊敏性と拡張性を実現する構成可能なアーキテクチャで企業を強化する方法を説明します。 モジュール型サービス、API ファースト統合、管理 UI 拡張機能により、パフォーマンスを損なうことなくカスタマイズされたコマースエクスペリエンスを実現する方法を説明します。

* コンポーザビリティがモジュラーアーキテクチャ、ヘッドレスコマース、API ファースト統合をどのようにサポートするかについて説明します。
* 管理 UI 拡張機能と開発者ツールが、カスタマイズとワークフローの機能強化を効率化する仕組みを説明します。
