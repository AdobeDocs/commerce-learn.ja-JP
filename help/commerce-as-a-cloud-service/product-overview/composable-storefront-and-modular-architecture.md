# 構成可能な柔軟なコマース体験

Adobe Commerce as a Cloud Serviceが、構成可能なアーキテクチャにより、俊敏性と拡張性を高めてどのようにビジネスを支援するのかをご覧ください。 モジュラーサービス、API ファーストの統合、管理UIの拡張性により、パフォーマンスを犠牲にすることなく、カスタマイズされたコマースエクスペリエンスを実現する方法をご確認ください。

* 管理UIの拡張性ツールと開発者ツールが、カスタマイズとワークフローの強化をどのように効率化するかをご確認ください。
