# 設定エクスペリエンス

Adobe Commerce as a Cloud Serviceを使用すると、セルフサービスの設定と安全なエンドポイントで、チームがストアフロントをすばやく立ち上げる方法を説明します。 エッジベースのアーキテクチャにより、超高速のパフォーマンスとスケーラビリティを実現する方法を説明します。

## このビデオの目的は誰ですか。

* E コマースマネージャー
* ソリューションアーキテクト
* デジタルトランスフォーメーションのリード

## ビデオコンテンツ

* セルフサービスダッシュボードを使用してAdobe Commerce ストアフロントを数分で設定し、起動する方法について説明します。
* Edge Delivery Servicesが優れたパフォーマンスと拡張性を実現する仕組みを説明します。
