# エンドツーエンドのショッピングジャーニーの実現

Adobe Commerce as a Cloud Serviceが、商品の検出からチェックアウト、購入後のサポートまで、シームレスなショッピングエクスペリエンスを強化する方法を説明します。 Live Search、製品レコメンデーション、セルフサービスダッシュボードなどの組み込みツールによって、コンバージョンを向上させ、運用オーバーヘッドを削減する方法を説明します。

## このビデオの目的は誰ですか。

* E コマースマネージャー
* カスタマーエクスペリエンススペシャリスト
* デジタルマーケティングマネージャー

## ビデオコンテンツ

* AI を活用した Live Search と製品レコメンデーションを使用して、ショッピングエクスペリエンスをパーソナライズする方法を説明します。
* セルフサービスアカウントツールを使用して、顧客の注文トラッキング、返品、再注文を簡素化する方法を説明します。
