# 一括データ移行ツールの概要

Dockerを準備し、Bulk Data Migration Toolにアクセスして、Adobe Commerce データを[!DNL Adobe Commerce as a Cloud Service] （[!DNL ACCS]）に移行します。

## この動画は誰のためのものでしょうか？

* ソリューションアーキテクト
* DevOps エンジニア
* バックエンド開発者

## ビデオコンテンツ

* 一括データ移行ツールの機能と制限について説明します。
* 開始する前に、Dockerがインストールされ、Docker エンジンが実行されていることを確認します。
* ツールパッケージを抽出し、正しいフォルダー権限を設定します。
* このシリーズの次のビデオで構成されている`.env.example` ファイルを確認します。
