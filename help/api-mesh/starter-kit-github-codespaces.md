# GitHub Codespacesを使用したAPI Mesh スターターキット

ここでは、Adobe Commerce API Mesh Starter Kitの概要を説明します。 ローカル設定なしでGitHub Codespacesを使用してAPI Meshを迅速に探索できるように設計されています。 リポジトリの設定、エンドポイントの設定、テスト用のGraphQL Yogaなどの組み込みツールの使用について説明します。

## この動画は誰のためのものでしょうか？

* フロントエンド開発者
* バックエンド/API開発者
* DevOps エンジニア

## ビデオコンテンツ

* GitHub Codespacesを使用して、ローカル環境を設定せずにAdobe Commerce API Meshを探索します。
* テスト用の組み込みツール：シームレスな開発のために、事前設定済みのGraphQLサーバー（Yoga）と自動サーバー再起動が含まれています。
* 実用的な学習：スターターキットのクローンを作成し、エンドポイントを設定し、API応答をブラウザーで直接テストします。
