# SEO メタデータを管理

ストアには、検索エンジンによるサイトのインデックス作成を改善するために、キーワードリッチメタデータを入力できる場所が読み込まれます。 時間の経過と共に、メタデータを微調整して、顧客の購入パターンや好みをターゲットに設定できます。

- Web サイトマネージャー
- e コマースマーチャンダイザー
