# 注文ステータスの管理

顧客が注文すると、販売注文が取引の一時的なレコードとして作成されます。 支払いが受け取られるまで、販売注文のステータスは&#x200B;_保留中_&#x200B;です。 「保留中」ステータスでは、支払が受け取られ、請求書が生成されるまで、注文を編集またはキャンセルできます。 このビデオでは、注文状況とその詳細を確認する方法と、必要に応じて注文の状況を変更する方法について説明します。
