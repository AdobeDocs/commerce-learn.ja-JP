# PhpStorm でのアップグレード互換性ツールの使用

アップグレード互換性ツール (UCT) は、現在のバージョンとターゲットアップグレードバージョン間の非互換性を数分で分析する無料のツールです。 PhpStorm プラグインを使用すると、このツールをさらに簡単に使用できます。

このビデオでは、

- アップグレード互換性ツールとは
- PhpStorm プラグインでのMagento
- HTMLの結果にアクセスし、モジュールを重大度と非互換性の数で視覚化する方法
