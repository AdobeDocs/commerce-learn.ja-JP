# PHP Storm でのアップグレード互換性ツールの使用

{{only-for-on-prem-commerce-cloud}}

このビデオの内容は次のとおりです。

- アップグレード互換性ツールとは
- Magento PHP Storm プラグインでの使用方法
- HTMLの結果にアクセスして、モジュールの重大度と非互換性の数で視覚化する方法
