# PHP Storm でのアップグレード互換性ツールの使用

アップグレード互換性ツール（UCT）は、現在のバージョンと対象とするアップグレードバージョンの間の非互換性をわずか数分で分析する、無料のツールです。 PHP Storm プラグインを使用すると、このツールがさらに使いやすくなります。
