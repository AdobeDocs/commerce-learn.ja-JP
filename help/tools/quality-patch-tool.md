# クォリティパッチツールの使用

品質パッチツールは、Adobe CommerceおよびMagento Open Sourceの品質パッチを提供するコマンドラインツールです。 次の操作が可能です。

- 最新の品質パッチに関する一般情報を表示
- 品質パッチの適用
- 適用済みのパッチを元に戻す

Adobe CommerceのサポートチームとMagento Open Sourceコミュニティは、品質パッチを開発します。
