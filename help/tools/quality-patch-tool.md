# Quality Patch Tool の使用

Quality Patch Tool は、Adobe CommerceとMagento Open Sourceのクオリティパッチを提供するコマンドラインツールです。 次の操作が可能です。

- 最新の品質パッチに関する一般情報の表示
- 品質向上パッチの適用
- 適用されたパッチを元に戻す

Adobe Commerce サポートチームとMagento Open Sourceコミュニティが、高品質のパッチを開発します。
