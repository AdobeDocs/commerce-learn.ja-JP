# 新しいページの作成

1 つのパラメーターで json を返すページを作成します。

## このビデオは誰のためのものですか。

- 開発者

## ページを追加する手順

- モジュールの作成
- **routes.xml** ファイルを追加します
- コントローラ（アクション）ファイルを追加する

## モジュールの作成手順

- モジュールフォルダーの作成
- `etc/module.xml` ファイルを作成します。
- `registration.php` ファイルを作成します。
- `bin/magento setup:upgrade` スクリプトを実行して新しいモジュールをインストールします
- モジュールが動作していることを確認します。
