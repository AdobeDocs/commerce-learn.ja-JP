# 依存関係の挿入の例

依存関係の挿入とは、オブジェクト A がその依存関係を提供する外部オブジェクト B に対して、その依存関係を宣言できる設計パターンです。 A で宣言された依存関係は通常、クラスインターフェイスであり、B が提供する依存関係は、これらのインターフェイスに対する具体的な実装です。
