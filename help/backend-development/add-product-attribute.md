# 製品属性の作成

製品属性の追加は、 [!DNL Commerce]. 属性は、製品に関連する多くの実用的なタスクを解決する強力な方法です。 製品に dropdown-type 属性を追加する簡単なプロセスがあります。

このビデオでは、

- 次の値を指定して、clothing_material という属性を追加します。綿、革、絹、デニム、毛皮、羊毛
- この属性を製品表示ページに太字のテキストで表示
- デフォルト属性セットに割り当て、制限を追加します。
- 新しい属性を追加
