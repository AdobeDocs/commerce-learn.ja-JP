# 依存関係の挿入の例

依存関係の挿入とは、オブジェクト A がその依存関係を提供する外部オブジェクト B に対して、その依存関係を宣言できる設計パターンです。 A で宣言される依存関係は通常、クラスインターフェイスで、依存関係 B が提供するインターフェイスは具体的な実装です。
