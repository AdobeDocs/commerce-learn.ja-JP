# 依存関係の挿入の例

依存関係の挿入は、オブジェクト A が依存関係を提供する外部オブジェクト B に対して依存関係を宣言できるようにする設計パターンです。 A で宣言される依存関係は通常、クラスインターフェイスであり、依存関係 B はそれらのインターフェイスの具体的な実装を提供します。
