# 依存関係インジェクションの例

依存関係インジェクションは、オブジェクト Aがその依存関係を提供する外部オブジェクト Bにその依存関係を宣言できるようにするデザインパターンです。 Aによって宣言される依存関係は通常、クラスインターフェイスであり、Bが提供する依存関係はそれらのインターフェイスの具体的な実装です。
