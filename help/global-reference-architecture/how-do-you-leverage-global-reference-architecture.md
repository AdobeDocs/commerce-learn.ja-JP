# グローバル参照アーキテクチャの活用

GRAを活用する方法について説明します。 GRAの導入を成功させるために、Adobeのエキスパートは、プロジェクトのさまざまなフェーズについて話し合い、発見フェーズについて詳しく説明します。 QA リソースについて説明し、GRAの実装にとってQA リソースが貴重なリソースである理由を説明します。

## オーディエンス

* 複数のブランドを持つ大規模プロジェクト
