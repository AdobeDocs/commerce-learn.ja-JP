# 製品属性の作成

Adobe Commerceでは、商品、顧客、商品レーティングに関連する情報を管理するために属性を使用します。 属性とは、フォーム、レポート、製品レビュー、階層化されたナビゲーション、メールメッセージなどのフィールドとして表示される情報の一部です。 商品カタログの構成要素であり、商品の特定の特徴を表します。

## この動画は誰のためのものでしょうか？

* web サイトマネージャー
* コマースマーチャンダイジング
