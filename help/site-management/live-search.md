# [!DNL Live Search] の概要

Adobe Commerceの [!DNL Live Search] について簡単に説明します。 Adobe Senseiを活用した [!DNL Live Search] は、人工知能と機械学習アルゴリズムを使用して、集計された訪問者データのディープ分析を実行します。
