# [!DNL Live Search]の概要

Adobe Commerceの[!DNL Live Search]の簡単な紹介です。 Adobe AIを活用した[!DNL Live Search]では、人工知能とマシンラーニング アルゴリズムを使用して、集約された訪問者データの詳細な分析を実行します。
