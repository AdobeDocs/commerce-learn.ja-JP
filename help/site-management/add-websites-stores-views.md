# Web サイト、ストア、ストア表示の追加

すべてのAdobe CommerceとMagento Open Sourceのインストールでは、web サイト、ストア、ストアビューの階層をサポートしています。 ビジネスのニーズに応じて、この階層を構築および拡張します。

## このビデオの目的は誰ですか。

- ストア管理者
- Web サイト管理者
