# 二段階認証

データへの不正なアクセスを防ぐには、管理者へのログインを試みるすべてのユーザーが認証プロセスを完了して、自分の ID を検証する必要があります。 Adobe CommerceとMagento Open Sourceは、複数のプロバイダーからの 2 要素認証 (2FA) 方式をサポートしています。

## このビデオは誰のためのものですか？

- ストア管理者
- Web サイトマネージャー
