# 二要素認証

データへの不正アクセスを防ぐには、管理者にログインしようとするすべてのユーザーが、認証プロセスを完了して ID を確認する必要があります。 Adobe CommerceとMagento Open Sourceは、複数のプロバイダーの二要素認証（2FA）方式をサポートしています。

## このビデオの目的は誰ですか。

- ストア管理者
- Web サイト管理者
