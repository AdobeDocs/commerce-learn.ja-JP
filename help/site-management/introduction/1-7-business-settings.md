# [!DNL Commerce] ビジネスユーザー向け — ストア設定

このビデオでは、

- ロケール設定
- ストア情報
- セキュア URL と非セキュア URL
- Web サイトのタイトル、ロゴ、著作権情報
- 通信メールのロゴを保存
- メールアドレスを保存
- 請求書、出荷、クレジットメモのロゴ
- 通貨オプション
- 通貨レート

## このビデオは誰のためのものですか？

- ビジネスユーザー
