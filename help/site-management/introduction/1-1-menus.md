# [!DNL Commerce] （ビジネスユーザー向け） [!DNL Admin] メニュー

The [!DNL Commerce Admin] はパスワードで保護されたバックオフィスで、製品とプロモーションの設定、顧客の注文の処理、 [!DNL Commerce] ビジネスニーズに合わせて保存します。

このビデオでは、

- 管理者メニューのレビュー
