# 通貨および税務ルールの管理

税ルールには、製品区分、顧客区分および税率の組み合わせが組み込まれます。 各顧客は顧客クラスに割り当てられ、各製品は製品クラスに割り当てられます。 [!DNL Commerce] 各顧客の買い物かごを分析し、顧客と製品の区分、および地域（配送先住所、請求先住所または発送元に基づいて）に基づいて適切な税金を計算します。
