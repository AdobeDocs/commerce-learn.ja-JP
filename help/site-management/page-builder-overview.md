# [!DNL Page Builder] の概要

[!DNL Page Builder]を使用すると、視覚的なstorytellingを強化し、顧客エンゲージメントとロイヤルティを促進するカスタムレイアウトを使用した、豊富なコンテンツを含むページを簡単に作成できます。

2.4.3 リリース以降、[!DNL Page Builder]はAdobe CommerceとMagento Open Sourceの両方でデフォルトで使用できます。 Adobe Commerceの[!DNL Page Builder]を使用すると、ステージングとプレビュー、パーソナライズされたコンテンツ、商品レコメンデーションを組み込むことができ、その他の機能も追加されます。

このビデオでは、次の方法について説明します。

* 既存ページの編集
* 新しいページの編集
* 既存のページへの製品の追加
* 行と列の追加と書式設定
* 列と行にメディアとエレメントを追加する
* スライダーの追加と書式設定
