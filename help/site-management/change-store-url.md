# ストア URL の変更

Adobe CommerceまたはMagento Open Sourceのインストール内の各 web サイトには、ストアフロントに割り当てられたベース URL と、管理者に割り当てられた別の URL があります。 いくつかの簡単な手順に従って、ストアのベース URL を変更できます。

## このビデオの目的は誰ですか。

- ストア管理者
- Web サイト管理者
