# ストア URLの変更

Adobe CommerceまたはMagento Open Source インストールの各web サイトには、ストアフロントに割り当てられたベース URLと、管理者に割り当てられた別のURLがあります。 ストアのベース URLを変更するには、いくつかの簡単な手順に従います。

## この動画は誰のためのものでしょうか？

* ストア管理者
* web サイト管理者
