# カテゴリの作成

商品をカタログに追加する前に、カタログの基本的なカテゴリ構造を確立する必要があります。 製品は、0個以上のカテゴリに割り当てることができます。 商品をカタログに追加する前に、通常、カテゴリーを事前に作成します。 ただし、製品の作成中に「その場で」カテゴリを追加することもできます。
