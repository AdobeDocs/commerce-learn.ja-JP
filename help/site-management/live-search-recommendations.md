# レコメンデーション用[!DNL Live Search]

{{ee-feature}}

Adobe Commerceの[!DNL Live Search]は、Adobe Commerceの高速で非常に関連性の高い直感的な検索体験を追加料金なしで提供します。 Adobe AIを活用した[!DNL Live Search]では、人工知能とマシンラーニング アルゴリズムを使用して、集約された訪問者データの詳細な分析を実行します。 このデータをAdobe Commerceカタログと組み合わせることで、非常に魅力的で関連性の高い、パーソナライズされたショッピング体験を実現します。 このビデオでは、ライブサーチを使用して商品のレコメンデーションを強化する方法を説明し、継続的なウェビナーシリーズの一部である「Commerce quick wins」を紹介します。
