# SEOのURL書き換えを管理

URL書き換えツールを使用すると、商品、カテゴリ、またはCMS ページに関連付けられている任意のURLを変更できます。 書き換えが有効になると、前のURLを指すリンクはすべて新しいアドレスにリダイレクトされます。

## この動画は誰のためのものでしょうか？

* web サイトマネージャー
* コマースマーチャンダイジング
* コマースマーケター
