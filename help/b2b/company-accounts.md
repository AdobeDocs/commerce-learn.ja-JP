# 会社アカウントの管理

{{b2b-feature}}

B2B 商人の場合、 _[!UICONTROL Companies]_ページには、現在のすべての会社アカウントとそのステータスが一覧表示されます。 ここで会社のアカウントを管理でき、承認の保留中のリクエストがリストの上部に表示されます。 標準の作業環境コントロールを使用して、リストのフィルタ、列のレイアウトの変更、ビューの保存、データの書き出しを行うことができます。

## このビデオは誰のためのものですか？

- Web サイトマネージャー
- セールス担当者
- カスタマーサービス担当者
