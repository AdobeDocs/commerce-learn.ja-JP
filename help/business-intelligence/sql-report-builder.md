# SQL REPORT BUILDER

Adobe Commerce Intelligence SQL Report Builderなら、データウェアハウスに対してクエリを実行し、結果を確認して、チャートに変換できます。 SQLを使用してレポートを作成する場合、作成する列を調整するために更新サイクルを待つことはありません。 エラーが発生した場合は、クエリを編集して再実行し、出力が期待するものと一致するようにします。

>[!IMPORTANT]
>
> SQL Report Builderを使用するには、管理者権限が必要です。

>[!TIP]
>
> Commerce Intelligenceを初めて利用する場合は、[Adobe Commerce Intelligence ユーザーガイド &#x200B;](https://experienceleague.adobe.com/docs/commerce-business-intelligence/mbi/guide-overview.html?lang=ja){target="_blank"}で、データを集計、分析、視覚化して、より強力なビジネスインサイトを獲得する方法を説明します。

## この動画は誰のためのものでしょうか？

* ビジネスユーザー
* データアナリスト

## ビデオコンテンツ

* SQL Report Builderを開き、データウェアハウスに対してクエリを実行します。
* クエリ結果からビジュアライゼーションを作成します。
* レポートをチームのダッシュボードに保存します。
