# Adobe Commerce Data Connectionのフルデモ

Adobe Commerce Data Connectionの詳細。

## 学習すること

Adobe Commerce Data Connectionの使用方法と基本的な設定について説明します。 このデモでは、Adobe CommerceとほかのAdobe サービスを連携することで、顧客の全体像を把握する方法を示します。

## この動画は誰のためのものでしょうか？

* Adobe Commerceを初めて利用する開発者やストアオーナーは、Adobe Data Connectionを使用した大規模なパーソナライゼーションについて詳しく知りたいと考えています。

## ビデオコンテンツ

* Adobe Commerceの行動データを他のDX製品（AJO、Analytics、Target、Real-Time CDPなど）に送信し、セグメントの意思決定、プロファイルの強化、カスタマージャーニー全体でのパーソナライズされたエクスペリエンスを実現。
* Real-Time CDPのオーディエンス ブラウザーデータ、サーバーサイドのデータ（注文履歴を含む）、その他のソースからのデータ（ロイヤルティデータ）にもとづいて、リアルタイムでセグメントを更新および変更する。
* Real-Time CDPのオーディエンスを活用して、Adobe Commerceのカート価格ルールとダイナミックブロックの両方をパーソナライズ。
