# Adobe Commerce Data Connection の完全なデモ

Adobeのコマースオプションについて説明します。 主な機能には何が含まれるか、どのようにホストされているか、各モデルが最も適しているユーザーについて説明します。 すべてを自分自身で実行する（オンプレミス）、Adobeが管理するクラウドインフラストラクチャ（PaaS）を使用する、完全に管理された SaaS アプローチを採用する、既存のバックエンドを維持しながらCommerce Optimizerでストアフロントレイヤーのみを最新化する、などの違いを学習します。

## このビデオの目的は誰ですか。

* コマース担当 VP/ディレクター
* 技術的リーダーシップ
* DevOps/Platform エンジニア
* マーケティングテクノロジーマネージャー

## ビデオコンテンツ

* Adobe Commerce デプロイメントモデル（オンプレミス、クラウド、SaaS、Optimizer）を比較
* 各オプションでインフラストラクチャとアプリケーションを管理する担当者を説明します
* バックエンドを置き換えずにストアフロントを最新化するタイミングを理解する
* DevOps リソース、スケーラビリティ、ビジネス目標に基づいて最適なものを特定します
