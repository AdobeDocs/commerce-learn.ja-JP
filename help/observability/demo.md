# Observability デモ

Adobe Commerceのオブザーバビリティのライブデモをご覧ください。 このデモでは、webhookとイベントワークフローがさまざまなサービスを接続して、注文追跡と問題解決を容易にする方法を示します。 注文の配送方法、エラーの修正方法、プロセスのチェック方法を確認します。  同期フローと非同期フローについて説明します。 また、GrafanaとNew Relicでトレースとログがどのように表示されるかを説明し、トラブルシューティングを行います。 スローなスポットを見つけ、システムをスムーズに動作させるために、相関関係が重要な理由をご確認ください。

## この動画は誰のためのものでしょうか？

* Adobe Commerce Developer
* ソリューションアーキテクト
* テクニカルプロダクトマネージャー

## ビデオコンテンツ

* 実際の注文ワークフローでAdobe Commerceの観測可能性を示す
* トレーサビリティのためのWebhookとイベントの相関関係について説明します
* ログとトレースのGrafanaおよびNew Relicとの統合を示します
