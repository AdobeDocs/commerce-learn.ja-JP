# Observability overview

最新のAdobe Commerce アーキテクチャにおいて、一元化されたオブザーバビリティが不可欠な理由をご確認ください。 拡張性が、独立したアプリ、webhook、UI拡張機能を備えた構成可能なモデルに進化するにつれ、マーチャントは複雑な実行フローを追跡するという課題に直面します。 統合された可観測性レイヤーがサービス間のシグナルを相関させ、先見的な最適化を可能にし、迅速で信頼性の高いチェックアウト体験を保証する方法を学びます。

## この動画は誰のためのものでしょうか？

* Adobe Commerce Developer

* テクニカルプロダクトマネージャー

## ビデオコンテンツ

* コンポーザブルモデルで複雑さが増す：独立したアプリやwebhookには単一のコンテキストが欠如しています。
* 一元化された可観測性が重要：すべてのコンポーネントにわたってログと指標を関連付けます。
* チェックアウトパフォーマンスを高める：ショッピングをスムーズに保つために、ボトルネックをすばやく特定します。
