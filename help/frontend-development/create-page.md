# 新しいページを作成

1 つのパラメーターで json を返すページを作成します。

## このビデオの目的は誰ですか。

- 開発者

## ページを追加する手順

- モジュールの作成
- **routes.xml** ファイルの追加
- コントローラ（アクション）ファイルを追加する

## モジュールの作成手順

- モジュールフォルダーの作成
- `etc/module.xml` ファイルの作成
- `registration.php` ファイルの作成
- `bin/magento setup:upgrade` スクリプトを実行して、新しいモジュールをインストールします
- モジュールが動作していることを確認します
