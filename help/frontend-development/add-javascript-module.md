# JavaScript モジュールの追加

{{only-for-on-prem-commerce-cloud}}

ストアフロントで「Hello World」の挨拶を表示するシンプルなJavaScript モジュールを開発する方法を説明します。

## この動画は誰のためのものでしょうか？

* 開発者

## 手順

1. モジュールを作成します。
2. `requirejs-config.js`とJavaScript モジュールファイルを追加します。
3. モジュールのテンプレートを読み込むレイアウト更新を作成します。
4. テンプレートファイルを作成します。
5. モジュールを登録し、ストアフロントでテストします。

## ビデオコンテンツ

* モジュールとRequireJS設定を設定します。
* レイアウトの更新とテンプレートを通じてスクリプトを接続します。
* ページ上の挨拶を確認します。
