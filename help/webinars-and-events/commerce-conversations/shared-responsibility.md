# 共通の責任

Adobe Commerceを安全に保つ上で、Adobeと販売者が責任を分担する方法をご紹介します。 Adobeでは、プラットフォームのコードの管理、重要な標準へのコンプライアンスの確保、サーバーの保護など、セキュリティの中核的な側面を担当します。 Adobe Commerce Cloud プロジェクトを実行するために、Adobeとマーチャントが認識しておく必要があることについて説明します。

## この動画は誰のためのものでしょうか？

* IT セキュリティの専門家
* E コマース責任者
* 開発者

## ビデオコンテンツ {#video-content}

* セキュリティの責任の共有：Adobeは、プラットフォームのコードのメンテナンスやコンプライアンスの確保などの主要なセキュリティタスクを処理します。一方、マーチャントは、独自のコードや統合を保護する必要があります。
* インシデントへの対応：セキュリティ警告には、電子メールや管理者通知などのさまざまなチャネルから迅速に対応する必要があります。
* ベストプラクティス：Adobeでは、二段階認証の有効化、安全な管理者アクセスの設定、最小権限の原則に従うなど、マーチャント向けのベストプラクティスをお勧めします。
