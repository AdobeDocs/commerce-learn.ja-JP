# New Relic アラート通知の設定

New Relicでアラート、ポリシー、ワークフロー、宛先を検索する方法について説明します。 既存のポリシーにメールを追加することがいかに簡単かをご確認ください。

## この動画は誰のためのものでしょうか？

* New Relicの既存のアラートに新しいメールを追加する必要がある開発者とチーム

## ビデオコンテンツ {#video-content}

* New Relic パネルのアラートの管理場所を確認します
* サポートチームがAdobe Commerce Cloud プロジェクト用に作成したアラートの一部をご覧ください
* アラート通知、ポリシー、ワークフローについて詳しく見る
* 宛先を保存し、新しい電子メールを検証することがポリシーに追加されました
