# Salesforce Commerce cloud starter kitについて詳しく見る

Salesforce Commerce Cloud（SFCC）とAdobe Commerce Optimizerを統合するために設計されたCommerce Optimizer Salesforce Commerce Cloud コネクタについて説明します。 この統合により、完全なリプラットフォームがなくても、従来のe コマースシステムを近代化し、製品の発見やリアルタイムのカタログ同期などのAIを活用した機能を活用することが可能になります。 Adobe Exchangeで利用できるコネクタは、カタログと価格表の同期、複数の製品タイプ、スケジュール済みのデータとオンデマンドのデータ同期の両方をサポートしています。 Adobe App Builder上に構築され、特定のビジネスニーズに合わせて拡張できる、カスタマイズ可能なスターターキットとして機能します。
