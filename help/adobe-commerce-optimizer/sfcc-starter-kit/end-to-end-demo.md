# Salesforce Commerce Cloud starter kitのエンドツーエンドのデモ

Salesforce CommerceとCommerce Optimizerの間で、スターターキットを使用して完全かつ差分の同期を正常に設定し、実行するエンドツーエンドのデモをご覧ください。 ランタイムログとイベントトレースを通じて同期の成功を確認し、ライブストアフロントで結果を検証する方法を説明します。 差分同期機能の仕組みをご紹介します。 また、Salesforce Commerce Cloudカタログの商品情報や価格の変更などのステップも説明します。Commerce Optimizerはこれらの変更を自動的に検出して更新し、ストアフロントに即座に反映します。

## この動画は誰のためのものでしょうか？

* Commerce Solution Architect
* テクニカルマーケティングエンジニア
* e コマースプラットフォーム管理者

## ビデオコンテンツ

* App Builderとスターターキットを使用して、Salesforce Commerce CloudとCommerce Optimizer間の完全な同期を設定および実行します。
* 完全同期を開始し、ランタイムログとイベントトレースを通じて成功を確認します。
* Salesforce Commerce Cloud カタログ内の商品の詳細と価格の変更は、Delta Sync プロセスを介してCommerce Optimizerで自動的に検出および更新されます。
* リアルタイムで更新され、システム間のシームレスなデータフローと同期を実証します。
