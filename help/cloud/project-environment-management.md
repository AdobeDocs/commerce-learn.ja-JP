# Adobe Commerce Cloudのプロジェクトと環境管理

Adobe Commerce Cloudを使用してプロジェクトと環境を管理する方法を説明します。 適切な電子メールとMAG IDでログインし、プロファイルを設定し、SSH キーを追加し、実稼動やステージングなどのさまざまな環境のレイアウトを理解する方法を示します。 また、システムの適切な部分へのアクセスを許可する方法、変更を加えた後に再展開することが重要な理由、クラウドアクセスとバックオフィスへのアクセスの違いについても説明します。 ここでの目的は、ユーザーがプロジェクトを整理して安全に運営し、スムーズに運営できるようにすることです。

## このビデオの対象

* クラウド運用エンジニア
* Adobe Commerce Cloud管理者
* テクニカルプロジェクトマネージャー

## 学習すること

* ログインしてプロファイルを設定
* 環境とデプロイメントの管理
* ユーザーのアクセスと権限の制御
