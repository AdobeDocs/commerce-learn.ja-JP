# Adobe Commerceの管理者ロール

Adobe IDまたはFederated IDを使用してAdobe Commerce管理ポータルにログインするプロセスと、ストア管理者がロールベースのアクセスを作成および管理する方法について詳しく説明します。 顧客の見積もりとテンプレートの管理を担当する「販売者」役割など、特定の権限を持つ新しい役割を作成する手順を理解します。 これらの役割にユーザーを割り当て、適切なアクセスレベルを確保する方法について説明します。また、新しく作成した役割にユーザーを割り当て、割り当てを検証する方法についても説明します。

## このビデオの対象

* アクセスと権限の管理を担当する管理者とビジネスユーザーを保存します。
* 特定の権限を持つ役割の作成と、これらの役割へのユーザーの割り当てに関する手順を必要とする新しいストア所有者
* Adobe Commerceのプロジェクトでは、Adobe Commerce管理者で実行される管理タスクを安全かつ効率的に管理することに関心があります。

## ビデオコンテンツ

* 最新のログイン方法について説明します。
* 役割の作成と管理についての理解を深めます
* ユーザーの割り当てとアクセスレベルについて。 &#x200B;
