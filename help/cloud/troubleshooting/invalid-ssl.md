# 期限切れのSSL証明書

期限切れのssl証明書を識別する方法、およびAdobe Commerce Cloud統合用またはその他の開発環境用にssl証明書を再生成する方法について説明します。

## このビデオの対象

* Adobe Commerce Cloudとの統合やその他の開発ブランチを使用して、テストを行う開発者。
* Adobe Commerce Cloudの新しいチームメイトで、Luma ストアフロントをテストに使用している人。

## ビデオコンテンツ

* 無効なssl証明書を検出する方法を説明します。
* プロセス中に新しい証明書を作成するタイミングを特定します。
* 新しい証明書が期待どおりに動作していることを検証します。 &#x200B;
