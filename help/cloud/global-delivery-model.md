# Adobe Commerceのグローバル配信モデル

Adobe Commerceは、AWSとAzureで利用可能なハイブリッドクラウド e コマースプラットフォームで、カート、チェックアウト、在庫管理などのコア機能を提供します。 ライブ検索や商品レコメンデーションなど、AIを活用したサービスを特徴とし、広範なAPI サポートを備えた安全でスケーラブルなインフラストラクチャを提供します。 このプラットフォームには、API MeshやApp Builderなどの開発者ツールが含まれており、高いセキュリティとコンプライアンス基準を遵守しながら、Commerce on CloudとAdobe Commerce Managed Servicesの2つのホスティングオプションを提供しています。

## このビデオの対象

* Adobe Commerceなら、コマース基盤の構築や強化に検討しているあらゆる規模の企業に最適なツールです。
* 拡張性、安全性、カスタマイズ性に優れたプラットフォームを求める小売企業、卸売企業、企業は、顧客体験と業務効率を最適化するために、AIを活用した高度な機能を備えた機能を利用できます。
* 開発者とIT部門は、コマースアプリケーションを構築、管理するためのクラウドネイティブなツールとAPIの堅牢なセットを提供するコマースオプションを調査しています。

## ビデオコンテンツ

* 拡張性とセキュリティに優れたインフラストラクチャ。  AWSおよびAzure上に構築された、完全に冗長なマルチクラウドインフラストラクチャにより、高いセキュリティとコンプライアンス基準を維持しながら、最適化されたパフォーマンス、高いレジリエンス、柔軟なスケーラビリティを実現する方法をご確認ください。
* AIを利用した高度なサービス。 ライブ検索、AIを活用した商品レコメンデーション、カタログサービスなどのAPI ファーストのインテリジェントなサービスについて理解を深め、顧客体験と業務効率を向上させることができます。
* 包括的な開発者向けツール： API MeshやApp Builderなど、シームレスな統合とカスタマイズが可能な、クラウドネイティブでサーバーレスな開発者向けツール群について説明し、コマース機能を拡張できます。
